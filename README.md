# 乐购商城-微服务后端工程
## 技术栈
- Spring Boot
- Spring Cloud
- FastDfs
- RabbitMQ
- MySQL
- Redis
- Canal

## 整体架构
- registry - 注册中心
- config - 配置中心
-     config-repo 本地配置文件
- legou-common - 工具类
- legou-core 通用基类,controller/service/dao 通用方法
- legou-item 商品工程
- legou-search 搜索工程
